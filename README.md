# microservice-governance
## 说明
本开源项目是《微服务治理：体系、架构及实践》一书的配套源码工程。
<br>
'声明' ：读者可以随意使用本源码，但由于本工程定位是“示例”，缺少全面的鲁棒性设计及完整测试，因此不建议直接在生产环境中使用。
